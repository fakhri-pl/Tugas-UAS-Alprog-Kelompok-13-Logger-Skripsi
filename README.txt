Program ini bertujuan untuk mencatat progress skripsi dalam bentuk log.
Terdapat fungsi penambahan log baru, edit log, hapus log, daftar log, pencarian data log, jadwal bimbingan, dan daftar jadwal bimbingan yang telah tersimpan.

Dalam program ini, terdapat fitur linked list, binary tree, fungsi dengan struct, array, dan address.
Berikut adalah link video implementasi program ini: https://www.youtube.com/watch?v=Omnozy7LtEw
